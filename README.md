# Arduino-M5StickC-HTU21DF
Temperature & Humiditiy Monitor Arduino Application for M5Stick

## Overview

<img width="80%" src="./img/M5StickCTempHumidity.png" />  

## Requirement

- M5Stick-C
- HTU21D-F
- Arduino IDE (Version: 1.8.5)

## Description

###  Pin connections

|M5Stick-C  |HTU21D-F  |Note  |
|---|---|---|
|GND  |GND  ||
|5V->  | VIN ||
|G0  | SDA ||
|G26  | SCL ||
